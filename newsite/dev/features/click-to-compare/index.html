Click to set the comparison column for hypothesis tests

Tables are shaded to show which other columns are higher or lower than the reference column for each row.

Jun 5, 2019

Tables in the app now let you set a column as the basis of comparison, shading the table to show which other columns are higher or lower than the reference column, and to what level of statistical significance. The test is a two-tailed t test comparing each column proportion in turn to the proportion in the reference column. In a multitable, the test is conducted between categories within each column variable, and not across column variables. See here for more details.
