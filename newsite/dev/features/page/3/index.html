Show value labels on graphs

Feb 13, 2020

Early Access users can now decide whether to show value labels for graphs and charts, both in Tables & Graphs mode and on dashboards. Value labels are these small numbers on graphs that show you the value represented by that bar/column/segment… With the new feature turned on, these value labels will be displayed on all graphs you create. With the feature turned off (the default), the user needs to hover over a chart element to see a tool-tip with this value....
